# shopease_project
ShopEase is a modern and responsive online shopping website built using HTML, CSS, Bootstrap, and JavaScript. It provides a seamless shopping experience for users to explore and purchase products like clothing, electronics, and accessories for men, women, and kids.

Features :
 Category-wise product listings (Men, Women, Kids, Electronics)
 Product detail pages with image zoom & hover effects
 Add to Cart and Buy Now functionality
 Order confirmation page with smooth UI
 Image carousel and zoom lens effect for product preview
 Modern and responsive UI with Bootstrap
 Fully responsive on all screen sizes
 Clean code structure for easy customization

Tech Stack :mm
HTML5
CSS3 / Bootstrap 5
JavaScript / jQuery
Font Awesome Icons
